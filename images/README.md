## Image Guidelines:

- **Format:** JPG or PNG
- **Quality:** High resolution, optimized for web
- **Size:** Keep files under 500KB for fast loading
- **Style:** Professional, modern, consistent color tones
- **Licensing:** Ensure you have rights to use all images

## Where to Get Professional Images:

- **Your own photos:** Best option for authenticity
- **Free stock photos:** 
  - Unsplash.com (currently used as fallbacks)
  - Pexels.com
  - Pixabay.com
- **Paid stock photos:**
  - Shutterstock.com
  - iStock.com
  - Adobe Stock

## How to Add Images:

1. Download or prepare your images
2. Rename them according to the filenames above
3. Place them in this folder (d:\PortfolioRey\Page\images\)
4. The website will automatically use your images
5. If an image is not found, it will use the Unsplash fallback

## Image Optimization Tips:

- Resize images to recommended dimensions
- Compress images using tools like TinyPNG or Squoosh
- Use JPG for photos, PNG for graphics with transparency
- Aim for 70-85% quality for JPG compression

**Note:** All images currently have fallback URLs to professional Unsplash images, so your portfolio will look great even without custom images. However, adding your own images will make it more personal and authentic.
